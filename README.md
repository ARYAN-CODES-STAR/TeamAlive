# TeamAlive

Keep your Microsoft Teams status **Available** while you stay focused.

A lightweight Python script that prevents Microsoft Teams from switching your status to **Away**.

---

## Requirements

- Python 3.x installed
- Microsoft Teams running(Already Available for Ready state)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/teamalive.git
```

Move into the project directory:

```bash
cd teamalive
```

---

## Run

Start the script:

```bash
python keep-alive.py
```

That's it! 🎉

As long as the script is running, it will keep your Teams status active.

To stop it, simply press:

```text
Ctrl + C
```

---

## Notes

- No installation required.
- No external dependencies.
- Runs locally on your machine.
- Does not modify Microsoft Teams.
